# Project2-cos161
A search engine created for COS161

Usage:
-Change "INSERT FILE PATH HERE" on line 20 of UserInterface.java to the full path to the data FILE
-Run UserInterface.java
-Wait for the program to load the file and create maps for term frequency, inverse document frequency, and title inverse document frequency
-When the program is ready, enter the number of results you would like the program to display
-Enter a search query
-When finished, type EXIT to close the program
